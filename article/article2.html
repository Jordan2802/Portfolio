Animation en css

les animations sont très simples à mettre en place. il faut la propriété "aniamtion" et "@keyframes". La propriété animation va permettre de caractérisé l'animation en lui donnant une durée, une direction, une boucle, un nom, ect.... le @keyframes va définir l'animation en elle même. par exemple, une rotation, une translation, un changement de couleur ect... voici le code css pour la rotation de l'image: 
.para1 .img{
    background: url("../img/wow.png");
    background-position: center;
    background-size: contain;
    background-repeat: no-repeat;
    animation: 4s linear 0s infinite rotation;
  }
  @keyframes rotation {
    0%{
      transform: rotate(0);
    }

    100%{
      transform: rotate(360deg);
    }
  }
On peut biensure cumuler les propriétés pour faire de plus grosses animations: 
.para2 .img p{
          position: absolute;
          font-size: 12em;
          color: #64dd17;
          font-weight: bolder;
          animation: 1.5s linear 0s infinite multianimate;
        }
        @keyframes multianimate {
          0%{
            transform: translate(0);
            font-size: 12em;
            color: #64dd17;
          }
          20%{
            transform: translate(10px);
            font-size: 10em;
            color: #76ff03;
          }
          40%{
            transform: translate(20px);
            font-size: 8em;
            color: #b2ff59;
          }
          60%{
            transform: translate(20px);
            font-size: 8em;
            color: #ccff90;
          }
          80%{
            transform: translate(10px);
            font-size: 10em;
            color: #b2ff59;
          }
          100%{
            transform: translate(0px);
            font-size: 12em;
            color: #64dd17;
          }
        }
